# 実験の結果
はじめに、正しいかどうかについては責任は取れません。

## 全体として
* 1はデフォルト
* 2はMaxChildを４倍にしたもの
* eventはmpmのeventで測った
* workerはmpmのworkerで測った
* preforkはmpmのpreforkで測った
* n は1000刻み　c は100刻みで6000 600まで計測している
* タグはfiatとmahjong(笑)で測った

## normal_*に関して
これはkeep-aliveをoffにして測ったもの

## keep_＊に関して
これはkeep-aliveをonにして測ったもの
